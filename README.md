# Beware the Dark Wizards

**Beware the Dark Wizards** is a riveting CLI adventure game written in C.

## Requirements

Compiling the app requires installation of `gcc`.

## Usage

Run the following commands in your terminal.

```sh
# Compile the code
gcc hello.c -o hello

# Launch the game
./hello
```
